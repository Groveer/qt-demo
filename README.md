# QtTest
qt demo and test
# Dependencies
## Build dependencies
* pkg-config
* cmake
* Qt5(>= 5.11.3) with modules:
    - qt5-base
## Runtime dependencies
# Installation
## Build from source code
1. Make sure you have installed all dependencies.

2. Build:
```
$ cd qt-demo
$ mkdir build
$ cd build
$ cmake ..
$ make
```

3. Install:
```
$ sudo make install
```
